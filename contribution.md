# 🤝 Contributing Guide

Thank you for contributing to this project.

This repository is designed to help beginners make their **first open-source contribution**.

---

# 📌 Contribution Rules

Please follow these rules carefully:

* Do not remove existing contributors
* Add only your own information
* Keep JSON formatting valid
* One profile per contributor
* Be respectful and professional

---

# 🧑‍💻 Steps to Contribute

### 1. Fork the Repository

Click the **Fork** button on the top right of this repository.

---

### 2. Clone Your Fork

```
git clone https://github.com/YOUR-USERNAME/REPOSITORY-NAME.git
```

---

### 3. Add Your Profile

Open the file:

```
contributors.json
```

Add your information using this format:

```json
{
  "name": "Your Name",
  "github": "your-github-username",
  "bio": "Short bio about you",
  "skills": ["React", "Node", "Python"]
}
```

Make sure the JSON structure remains valid.

---

### 4. Commit Your Changes

```
git add contributors.json
git commit -m "Added my contributor profile"
```

---

### 5. Push to Your Fork

```
git push origin main
```

---

### 6. Create a Pull Request

Go to your forked repository and click **New Pull Request**.

Our automated system will:

* Validate your JSON entry
* Run project checks
* Build the website

If everything passes, your Pull Request can be merged.

---

# 🎉 After Your PR is Merged

Once your Pull Request is accepted:

* Your profile will appear on the project website
* Your GitHub avatar will be displayed
* Your contribution will be recorded in the leaderboard

---

# 🙌 Thank You

Your contribution helps grow the open-source community.

Happy coding!
